# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-19

### Added
- Initial release of RPI Skills family
- 12 core skills: rpi-context, rpi-spec, rpi-research, rpi-grillme, rpi-plan, rpi-implement, rpi-review, rpi-handoff, rpi-pipeline
- 3 conversion helpers: rpi-plan-to-speckit-plan, rpi-spec-to-speckit-spec, rpi-tasks-to-speckit-tasks
- Agent Skills standard compliance
- Support for Pi, Claude Code, OpenAI Codex, and generic Agent Skills harnesses
- Automated validation script
- GitHub Actions CI validation
- Comprehensive README with usage guide and credits
